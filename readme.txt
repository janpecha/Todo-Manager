
Todo Manager
#####################

(c) Jan Pecha, 2010-2012 (http://janpecha.iunas.cz)

Vyžaduje:
	- PHP 5.2+	(kvůli třídě DateTime)
	- třídu SafeStream (třída přiložena ke zdrojovému kódu ve složce /cztodo)


Instalace
===========

Stáhněte zdrojový kód, rozbalte ho do nějaké složky, ke které má přístup webový
server (/var/html_public apod.) a spusťte pomocí webového prohlížeče (např.
http://localhost/todo-manager/).
